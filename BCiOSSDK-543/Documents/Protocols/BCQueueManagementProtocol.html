BCQueueManagementProtocol Protocol Reference

Conforms to NSObject
Declared in BCQueueManagementProtocol.h

Overview

The BCQueueManagement protocol specifies methods for managing a queue of items. Its features are inspired by Appleâ€™s AVQueuePlayer behavior.

The BCQueueManagement protocol supports basic queue operations such as adding and removing items from the queue, accessing the list of items in the queue and deterimining the current item.

Controlling Playback

  • – advanceToNextItem

    Advances playback to the next item in the queue. If there are no more videos then calling this method has no effect.

         required method
  • – advanceToItem:

    Advances the queue to the specified item. If the item is not found then calling this method has no effect.

         required method
  • – setActionAtItemEnd:

    Set the desired action the player should take when it reaches the end of the playing item.

setActionAtItemEnd:

Set the desired action the player should take when it reaches the end of the playing item.

- (void)setActionAtItemEnd:(int)actionAtItemEnd

Parameters

actionAtItemEnd

Indicates the desired action, value should be one of: – AVPlayerActionAtItemEndAdvance (0) – Advance to the next item if possible. – AVPlayerActionAtItemEndPause (1) – The player should pause. – AVPlayerActionAtItemEndNone (2) – The player should do nothing.
